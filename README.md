# 🚀 Smart Internship Tracker

A modern AI-powered internship tracking platform designed to help students organize applications, monitor progress, manage deadlines, and improve their internship search workflow.

## 📌 Overview

Smart Internship Tracker helps students keep track of internship applications from multiple companies in one place. Instead of using spreadsheets or notes, users can manage their entire internship journey through an intuitive dashboard.

## ✨ Features

* 📋 Track internship applications
* 📅 Application deadline management
* 📊 Progress tracking dashboard
* 🔍 Search and filter applications
* 🏢 Company information management
* 📈 Application status analytics
* 🎯 Internship opportunity organization
* 📱 Responsive and modern UI

## 🚀 Getting Started

### Prerequisites

* Node.js
* npm

### Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/smart-internship-tracker.git
```

Navigate to the project folder:

```bash
cd smart-internship-tracker
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```

## 🎯 Problem Statement

Students often struggle to manage multiple internship applications across different platforms. Important deadlines, application statuses, and interview schedules can become difficult to track.

This project provides a centralized solution for organizing and monitoring internship applications efficiently.
